# Espressif ESP32 WROOM-32 & ESP-WROVER-KIT

There are 2 images: 
 - One that _fits_ all EPS32 boards carrying an ESP32-WROOM-32 module, like the ESP32-DevKitC.
 - Another specific to ESP32-WROVER-KIT that include UI features.
